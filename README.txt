greg_kraus@ncsu.edu

Description:
This is a bookmarklet that will automatically download a Google Doc - a document, spreadsheet, or presentation - into its corresponding Microsoft Office format. Downloading a Google Doc involves manipulating the URL of the original document. The following are the formulas used for determining the correct download URL.

The JavaScript for this bookmarklet is loaded from the server each time the bookmarklet is activated.
